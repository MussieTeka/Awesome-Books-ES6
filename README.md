# 📖 [Awesome Books ES6] <a name="about-project"></a>

This is a JavaScript project that allows you to create a list of books and view them on a web page. The project uses modern ES6 syntax and features, as well as the Luxon library for handling dates and times.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

- Client: HTML, CSS, JavaScript
- Server: Not applicable
- Database: Not applicable

The project was built using the following technologies:

- HTML
- CSS
- JavaScript
- Luxon

### Key Features <a name="key-features"></a>

- **Allows the user to add new books to a list using a form**
- **Displays a list of all the books added by the user**
- **Allows the user to view contact informations**
- **Uses modern ES6 syntax and features**
- **Uses the Luxon library for handling dates and times**
- **Automatically updates the current time on the web page**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy of the project up and running, follow these steps.

### Prerequisites

In order to run this project, you need to have a web browser installed on your computer.

### Setup

Clone this repository to your desired folder by running the following command:

```sh
  git@github.com:MussieTeka/Awesome-Books-ES6.git
```

### Install

There are no installation steps required for this project.

### Usage

To run the project, open the index.html file in your web browser.

### Run tests

To run tests, run the following command:

```sh
 npx hint .
 npx stylelint "**/*.{css,scss}"
 npx eslint .
```

### Deployment

You can deploy this project using any web hosting service of your choice.
